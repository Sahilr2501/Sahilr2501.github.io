Resume API Project

A RESTful API serving professional resume data in JSON format

Overview

The Resume API is a Node.js/Express application that serves professional resume data in both JSON and HTML formats. This project demonstrates backend development skills and API design principles, providing a clean interface to access and display resume information programmatically.

Key Features

  • RESTful Endpoint: Simple GET endpoint that returns resume data in JSON format.
  • Formatted HTML Response: Option to receive the resume as a styled HTML page.
  • Data Persistence: Resume data stored in a JSON file for easy maintenance.
  • Responsive Design: HTML output is fully responsive and mobile-friendly.
  • Error Handling: Proper error handling for file operations and JSON parsing.

Technologies Used

Node.js Express.js JSON HTML5 Bootstrap
